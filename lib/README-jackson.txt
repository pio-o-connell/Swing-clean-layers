# Required JARs for Concordia Project

Download the following Jackson libraries and place them in the lib/ directory:

- jackson-databind-2.17.1.jar
- jackson-core-2.17.1.jar
- jackson-annotations-2.17.1.jar

You can download them from:
- https://repo1.maven.org/maven2/com/fasterxml/jackson/core/jackson-databind/2.17.1/
- https://repo1.maven.org/maven2/com/fasterxml/jackson/core/jackson-core/2.17.1/
- https://repo1.maven.org/maven2/com/fasterxml/jackson/core/jackson-annotations/2.17.1/

After downloading, re-run the build script.
